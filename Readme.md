# Репозиторій лабораторних робіт з дисципліни «Теорія і методи обчислювального інтелекту»

## Опис дисципліни
Цей репозиторій містить практичні та лабораторні роботи, виконані в межах вивчення курсу з обчислювального інтелекту. Основна увага приділяється проєктуванню, розробці та застосуванню інтелектуальних систем для розв'язання прикладних задач. Під час курсу розглядаються методи машинного навчання (з учителем та без учителя), алгоритми нечіткої логіки для роботи в умовах невизначеності, а також архітектури штучних нейронних мереж. Репозиторій демонструє повні цикли обробки даних: від очищення та підготовки датасетів до побудови, налаштування та об'єктивної оцінки моделей.

## Представлені роботи

### Звіт з аналізу та прогнозування енергоспоживання (Лабораторна робота №1)
Робота демонструє повний цикл обробки даних та побудови моделей машинного навчання. Включає вирішення задач регресії та класифікації, налаштування гіперпараметрів моделей та порівняння їхньої ефективності.

### Лабораторна робота №2: Кластеризація
Робота присвячена методам машинного навчання без учителя. Виконується розбиття даних на кластери для пошуку прихованих закономірностей та групування об'єктів за їхніми характеристиками.

### Лабораторна робота №3: Нечітка логіка та проєктування інтелектуального модуля
* **Мета:** Ознайомитися з принципами нечіткої логіки та її застосуванням для проєктування інтелектуального модуля, здатного приймати рішення в умовах невизначеності.
* **Опис:** Розроблено модуль для виявлення схожих записів у базі даних нерухомості Парижа. Система аналізує текстові дані (опис та адресу) для групування об'єктів, які ймовірно є ідентичними, попри орфографічні помилки чи різні формати запису.
* **Методи:** Для оцінки схожості використовуються алгоритми Jaro-Winkler (текстові перестановки), Soundex (фонетична схожість) та Damerau-Levenshtein (редакторська відстань).

### Лабораторна робота №4: Основи нейронних мереж (побудова та навчання перцептрона)
* **Мета:** Ознайомитись з поняттями штучних нейронних мереж, їхньою архітектурою, принципами навчання.
* **Опис:** Робота охоплює побудову та навчання нейронних мереж (від одношарового до багатошарового перцептрона) для вирішення задач бінарної класифікації. Продемонстровано застосування сучасних практик глибокого навчання, оптимізації та запобігання перенавчанню.

Лабораторна робота №5: Згорткові нейронні мережі (CNN) та Transfer Learning

**Мета:** Ознайомитися з архітектурою згорткових нейронних мереж та передовими методами перенесення знань (Transfer Learning) для розв'язання задач комп'ютерного зору (Computer Vision).
**Опис:** Робота присвячена побудові системи багатокласової класифікації зображень (на прикладі розпізнавання 20 різних франшиз/стилів аніме). Продемонстровано повний цикл підготовки візуальних даних. Виконано порівняльний аналіз трьох підходів: проєктування власної CNN з нуля (з використанням Global Average Pooling), а також застосування попередньо навченої на ImageNet моделі ResNet18 у режимах Feature Extraction (заморожена база) та Fine-Tuning (розморожування глибоких шарів).
